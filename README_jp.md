## 序章

:jack_o_lantern: 興味深い docker プロジェクトを共有します。 すべてのコンテンツは **オープン ソース**であり、プログラミングの基礎がなくても誰でも独自のサービスを構築できます。

```shell

# ありがとう
もしあなたが日本語と中国語を知っているなら、私のプロジェクトにすぐに参加できることを願っています。 ここであなたの助けが必要です。

```
